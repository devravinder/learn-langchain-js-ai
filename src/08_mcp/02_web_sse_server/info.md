

## Note:-
1. The SSE server should be in running state



## Usage
1. run inspector
   - `npx @modelcontextprotocol/inspector@latest`
   - or `pnpm run s:i`

2. open the inpsector UI in the browser
     - add 
       - Transport Type: `SSE`
       - url: `http://localhost:3001/sse`

    - connect
     - then navigate to `tools`
         - `list tools`
